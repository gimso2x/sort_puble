안녕하세요. SK telecom API Portal 입니다.

항상 SK telecom API Portal를 이용해주시는 고객님들께 감사 드립니다.
도메인 변경 공지사항 내용 중 Workplace 필수 재설정 내용을 한 번 더 전달해드립니다.

2018년 1월 4일(목) 22:00 ~ 1월 5일(금) 01:00 (총 3시간) 동안 도메인 변경으로 인한 시스템 점검 작업을
진행할 예정입니다.
서비스 개선 작업 이후 안정적인 서비스 반영을 위해 Workplace 재설정이 필요합니다.
이에 따라, 번거우시더라도 '필수항목'은 재설치 후 이용해 주시기 바랍니다.
 
 
  • 작업 일시 : 2018년 1월 4일(목) 22:00 ~ 2018년 1월 5일(금) 01:00 (총 3시간)
  • 작업 내용 : 도메인 변경으로 인한 시스템 점검 작업
  • (기존) www.tbizpoint.co.kr → (변경) www.sktelecomapi.com
  • 서비스 영향 : 작업시간 내 SK telecom API Portal 서비스 순단현상 발생 및
  • Workplace 로그인, 그룹웨어 서비스 중단 (메신저 로그인, 메일 송수신, 아웃룩 접속 포함)
 
 
[반영 이후 Workplace 필수 재설정 사항]
 
1. [필수] 그룹웨어 메신저 재설치
도메인이 변경됨에 따라 '그룹웨어 전체메뉴' > '메신저 다운로드'에서 메신저를 재설치하신 후 사용하시기
바랍니다.
2. [필수] MX 레코드 및 SPF 레코드 재등록
API Portal 도메인 MX 레코드 값 : gwspam.sktelecomapi.com / 호스트는 @
API Portal 도메인 SPF 레코드값 : "v=ip4:175.126.131.57 ip4:175.126.131.58 ip4:175.126.131.68
ip4:175.126.131.69 ~ all"
3. [필수] 메일 아웃룩 재설정
(기존) gwmail.tbizpoint.co.kr
(변경) gwmail.sktelecomapi.com
4. [필수] PC메일 보관함 재설치
PC메일보관함 백업 메뉴얼에 따라 백업을 필수로 진행하신 후, '그룹웨어 전체메뉴' > 'PC메일보관함 설치'에서
PC메일보관함을 재설치하신 후 사용하시기 바랍니다. (자세한 사항 메뉴얼 참고)
5. [필수] 오피스폴더 탐색기 재설치
도메인이 변경됨에 따라 '그룹웨어 전체메뉴' > '오피스폴더 탐색기'에서 재설치하신 후 사용하시기 바랍니다.

이 외 자세한 사항은 공지 사항을 확인해주시기 바랍니다.
 
 
궁금하신 점이나 보다 자세한 작업 관련 안내가 필요하신 경우 SK telecom API Portal 고객센터(1600-0990)로
문의 주시기 바랍니다.

감사합니다.
